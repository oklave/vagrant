#vagrant v1.0

This repository is intended to help with setting up vagrant and contains example vagrant files.

'''vagrant box add ubuntu/focal64 focal-server-cloudimg-amd64-vagrant.box'''
'''vagrant init ubuntu/focal64'''
'''vagrant up'''
